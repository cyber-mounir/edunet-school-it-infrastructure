# EduNet – School IT Infrastructure Project

## Overview
EduNet documents the design and implementation of a complete IT infrastructure
for a medium-sized school using Windows Server and enterprise networking.

## Scope
- Network design (Logical & Physical)
- VLAN segmentation and routing
- Windows Server: AD, DNS, File Server, GPO
- Basic firewall & security policies

## Project Structure
- docs/: Documentation and guides
- design/: Logical & Physical network design
- simulation/: Packet Tracer files
- systems/: Windows Server configuration
- security/: Security policies

## Status
Project completed – Version 1.0
