# Project Requirements

## Overview
Design and simulate a complete IT infrastructure for a medium-sized school.

## School Profile
- Type: Secondary School
- Floors: 4
- Students: ~400
- Staff: ~23

## Business Requirements
- Reliable internet access
- Secure internal network
- Centralized user management
- File sharing services
- Controlled access between departments

## Technical Requirements
- Network segmentation using VLANs
- Centralized authentication (Active Directory)
- Scalable IP addressing
- Basic firewall and access control policies
